# E-Commerce Fullstack (React + Express + MongoDB)

This archive contains a minimal full-stack e-commerce scaffold.

Backend: /backend
Frontend: /frontend

Instructions:
1. Backend:
   - cd backend
   - cp .env.example .env and fill MONGO_URI and JWT_SECRET
   - npm install
   - npm run seed   # optional to insert demo data
   - npm run dev

2. Frontend:
   - cd frontend
   - npm install
   - set REACT_APP_API_URL if backend not at default http://localhost:5000/api
   - npm start
